Methodology

  • Source: SEC EDGAR XBRL API. No Bloomberg, no consensus, no aggregators — the number on screen is the number in the filing.
  • YTD → standalone: Quarterly cash flow series rebuilt by differencing YTD 10-Q filings, the same way an analyst would manually, automated across decades.
  • Restatements preserved: Pre-restatement and post-restatement filings both kept; the chart knows which year came from which vintage.
  • Footnotes surfaced inline: Dotted-underlined accounting terms show their definition on hover where they appear, not buried in a glossary.
