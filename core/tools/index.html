Tools

Cross-cutting analytical tools: pair-trade backtest scaffolding, transmission-network maps, regime-conditional backtests, and a full indicator-comparison engine. Each runs against the full FRED + EDGAR dataset.

Pair Explorer →

Compare any two FRED series with rolling correlation, spread, and ratio views. Useful for testing macro pair-trade hypotheses.

Transmission Network →

Force-directed graph of how macro shocks propagate across rates, credit, equities, and the real economy. Click a node to see its first-order links.

Regime Backtest →

Backtest sector and factor returns conditional on the regime classification. Surfaces which exposures historically pay off in each cycle phase.

Compare Indicators →

Multi-series overlay across all 22 macro indicators. Choose a transformation (level, YoY, z-score) and align by date or event.
